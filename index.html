📱 폰 서명 등록 설정 방법 (처음 1회)

각자 폰으로 서명받아 내 구글 시트에 자동 저장되게 하는 설정입니다. 아래 순서대로 한 번만 하면 됩니다.
  1. 구글 시트 만들기 — 구글 드라이브에서 새 구글 시트를 하나 만듭니다.
  2. 코드 넣기 — 그 시트에서 확장 프로그램 → Apps Script를 열고, 기존 내용을 모두 지운 뒤 아래 ‘코드 복사’로 복사한 코드를 붙여넣고 저장합니다.
  3. 배포 — 배포 → 새 배포 → 유형 ‘웹 앱’, 실행 대상 ‘나’, 액세스 ‘모든 사용자’로 배포하고 권한을 허용합니다.
  4. 주소 복사 — 나온 웹 앱 URL(.../exec로 끝남)을 복사합니다.
  5. 붙여넣기 — 위 ‘① 내 구글시트 저장 주소(SCRIPT_URL)’ 칸에 붙여넣으면 끝!
↓ 붙여넣을 코드 (연수명별로 시트 탭 자동 분리) 

      
⚠️ 일부 학교·교육청 구글 계정은 앱스스크립트 배포가 막혀 있을 수 있어요. 그럴 땐 개인 구글 계정으로 만드세요. 이름·연락처·서명이 모이니, 구글 시트·드라이브 폴더 공유는 본인(담당자)만 보도록 두세요.

    

    

  








  

    
⚠️ 자료 저장 방식 안내

    

      이 사이트에서 입력·업로드한 모든 자료는 지금 사용하는 컴퓨터(브라우저)에만 저장됩니다. 인터넷 서버로 전송되지 않습니다.


      따라서 브라우저 기록(사이트 데이터·캐시)을 지우거나, 다른 컴퓨터·다른 브라우저에서 열면 저장한 자료가 사라집니다.
    

    
💾 중요한 내용은 상단 ‘백업 저장’으로 파일(JSON)로 받아두세요. 다른 컴퓨터로 옮길 때는 ‘백업 불러오기’를 이용하면 됩니다.
